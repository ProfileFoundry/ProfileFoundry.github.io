# ProfileFoundry.github.io

Simple GitHub Pages smoke test for the ProfileFoundry organization.

## Publish

Create a public repository named `ProfileFoundry.github.io` under the
`ProfileFoundry` organization, then push this repo:

```sh
git remote add origin git@github.com:ProfileFoundry/ProfileFoundry.github.io.git
git push -u origin main
```

Once GitHub Pages finishes deploying, the site should be available at:

```text
https://profilefoundry.github.io/
```
